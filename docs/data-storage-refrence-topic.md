# How to Manage Users’ Data in Bebbo? 

## A. Back Data Up 

1. Click the hamburger icon in the upper-left corner or the side bar of the app. 
2. Go to Settings, and then swipe down where “Transfer Data” is visible. 
3. Click **Back up your data** and choose where to save your data: **File Manager or Google Drive.**

- For File Manager: 

1. Choose where the file will be saved in your device (i.e, Downloads). 
2. Create a folder specifically for Bebbo files.
